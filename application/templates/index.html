Introduction

This is my Capstone Project for the Data Science semester of forward school. The graphs show the analysis of Unicorn Startups as of September 2022. The graphs were plotted using Plotly express. This dataset can be found on Kaggle. I extracted and cleaned this data using pandas. In business, a unicorn is a privately held startup company valued at over $1 BILLION DOLLARS.
